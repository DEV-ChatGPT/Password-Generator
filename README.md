# 🔐 Password-Generator

Generates a random secure password that you can save and export as a PDF

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-green.svg)

## ✨ Features

- 🎲 Generate random secure passwords
- 🔤 Customizable length (4-32 characters)
- 🔢 Toggle uppercase, lowercase, numbers & symbols
- 💾 Save passwords locally
- 🌐 Multi-language support (15 languages)
- 📄 Export all passwords as PDF

## 🚀 Demo

[Live Demo](https://yourusername.github.io/password-generator)

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- [jsPDF](https://github.com/parallax/jsPDF) - PDF generation

## 📖 Usage

1. Open `index.html` in your browser
2. Set your desired password length
3. Choose character types (uppercase, lowercase, numbers, symbols)
4. Click **Generate**
5. Enter an app/service name
6. Click **Save Password**
7. Export all saved passwords as PDF
